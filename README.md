# APP-WEB-4A
Este proyecto consiste en el desarrollo de un sitio web responsivo utilizando Bootstrap 5, como parte de una práctica académica orientada al fortalecimiento de habilidades en desarrollo Front-End. El objetivo principal es aplicar componentes esenciales como navbar, secciones en grid, tarjetas, footer y personalización de estilos, asegurando compatibilidad tanto en escritorio. El sitio incluye una sección original “Acerca del equipo”, integración de imágenes y modificación de al menos dos colores mediante clases de Bootstrap. El proyecto fue realizado por Oskar Hernández, Adriana Gonzalez, Heidy Garcia como desarrolladores y encargados de la documentación. Para su construcción se empleó la plantilla base “Starter Template Bootstrap 5”, junto con estilos personalizados para complementar el diseño. El repositorio contiene el archivo index.html y recursos necesarios para visualizar la página; para ejecutarlo únicamente es necesario clonar el repositorio y abrir el archivo en el navegador, sin requerir dependencias adicionales.
